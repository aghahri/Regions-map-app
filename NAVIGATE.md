# راهنمای رفتن به فولدر برنامه

## در Terminal:

### از دایرکتوری فعلی:
```bash
cd regions-map-app
```

### از هر جای دیگر:
```bash
cd /Users/akiokaviano/ai/code/regions-map-app
```

### یا از root:
```bash
cd ~/ai/code/regions-map-app
```

---

## در VS Code:

1. **File → Open Folder**
2. مسیر را وارد کنید: `/Users/akiokaviano/ai/code/regions-map-app`
3. یا از Explorer روی پوشه کلیک کنید

---

## بررسی اینکه در فولدر درست هستید:

```bash
pwd
# باید ببینید: /Users/akiokaviano/ai/code/regions-map-app

ls
# باید فایل‌های زیر را ببینید:
# - app.py
# - requirements.txt
# - Procfile
# - README.md
# - ...
```

---

## دستورات مفید:

```bash
# رفتن به فولدر برنامه
cd regions-map-app

# برگشت به فولدر قبلی
cd ..

# رفتن به root پروژه
cd /Users/akiokaviano/ai/code

# دیدن محتویات
ls -la

# دیدن مسیر فعلی
pwd
```
